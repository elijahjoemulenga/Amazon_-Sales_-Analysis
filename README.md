# Amazon Sales Analysis

## Project Overview
Analyzed fictional Amazon sales data to identify revenue trends, top sales reps, product categories, and sales channels. Key insights: Online channels generate more profit than retail, suggesting a focus on digital advertising.

## Tools Used
- **SQL** (MySQL Workbench): For querying and aggregating data (e.g., SUM(Revenue) by group).
- **Excel**: Data cleaning and initial exploration (Implementation of pivot tables).
- **Power BI**: Visualizations like bar charts for monthly revenue and pie charts for payments.

## Key Steps
1. **Data Cleaning**: Removed duplicates, standardized formats (e.g., dates, text) in Excel.
2. **SQL Analysis**: Ran queries to group by Sales_Rep, Product_Category, and Sales_Channel — ordered by profit DESC.
3. **Insights & Visuals**: Built Power BI dashboard showing trends (e.g., Q1–Q4 products by category) and recommendations.
